This code is written in java programming language using the following -

JAVA Swings - For setting up the GUI/Interface of the information page which can be used to input data and also to see a particular information.

MySQLConnector - To connect the java environment with the database using jdbc drivers.

XAMPP - The DATABASE where the information given in the info page is stored and can be retrieved.


Note - 

You will need the following softwares to run this code in your system with the database -

1)Eclipse IDE 

2)MySQLConnector (in order for the import java.sql.* and further jdbc drivers to work)

3) XAMPP - To access the database or create a database for yourself.
